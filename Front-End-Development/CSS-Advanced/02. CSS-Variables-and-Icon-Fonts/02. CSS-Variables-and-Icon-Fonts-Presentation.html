CSS Variables

CSS Variables

CSS variables are entities defined by CSS authors that contain specific values to be reused throughout a document. They are set using custom property notation (e.g., --main-color: black;) and are accessed using the var() function (e.g., color: var(--main-color);).

Icon Fonts

Icon Fonts

Icon Fonts - The Good

  • Infinitely scalable, and it works well both on standard and retina displays;
  • Can be styled with a CSS including text-shadow, gradient, color, :hover and :focus pseudo-classes and other properties. It also helps when you need to change the whole set of icons: just change a CSS rule — no necessity to redraw every icon;
  • Animatable;
  • Cross browser compatibility;
  • Fewer pictures — fewer requests to a server side.

Icon Fonts - The Bad

  • No ability to create multi-coloured icons: just single color or gradient;
  • Subject to anti-aliasing;

SVG

Scalable Vector Graphics (SVG) is an XML-based markup language for describing two dimensional based vector graphics. SVG is essentially to graphics what HTML is to text.

SVG Icons - The Good

  • Infinitely scalable, and it works well both on standard and retina displays;
  • Can be styled with a CSS
  • Cross browser compatibility;
  • Ability to have multiple colors in each icon
  • Ability to be part of the HTML document, thus reducing the number of HTTP requests
