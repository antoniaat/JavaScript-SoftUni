Sass

Sass

Sass is an extension of CSS that adds power and elegance to the basic language. It allows you to use variables, nested rules, mixins, inline imports, and more, all with a fully CSS-compatible syntax. Sass helps keep large stylesheets well-organized, and get small stylesheets up and running quickly, particularly with the help of the Compass style library.

Sass - Features

  • Fully CSS-compatible
  • Language extensions such as variables, nesting, and mixins
  • Many useful functions for manipulating colors and other values
  • Advanced features like control directives for libraries
  • Well-formatted, customizable output

Sass - Syntax

There are two syntaxes available for Sass.

  • SCSS (Sassy CSS) [.scss]
  • Indented syntax (or just "Sass") [.sass]

Using Sass

  • Ruby Sass - gem install sass
  • Node.js Sass - npm install node-sass (Bindings for LibSass, a C++ implementation)
  • Python Sass - pip install libsass (Bindings for LibSass, a C++ implementation)
  • Sass.js - Sass parser in JavaScript - https://github.com/medialize/sass.js

Less

Less

It's CSS, with just a little more.

  • SCSS (Sassy CSS) [.scss]
  • Indented syntax (or just "Sass") [.sass]

Less - Features

  • Evaluating values and conditional statements
  • Language extensions such as variables, nesting, and mixins
  • Many useful functions for manipulating colors and other values
  • Advanced features like control directives for libraries
  • Well-formatted, customizable output

Using Less

  • Node.js - npm install -g less
  • In the Browser - JavaScript - include less.js in your HTML
